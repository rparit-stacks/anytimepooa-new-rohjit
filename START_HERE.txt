Problem: Still getting "NOT_SUPPORTED" error

Solution 1: Manual Flag Setup
------------------------------
1. Close ALL Chrome/Brave windows
2. Open Chrome
3. Go to: chrome://flags/#unsafely-treat-insecure-origin-as-secure
4. Enable it
5. In the text box, add: http://192.168.1.11:3000
6. Click "Relaunch"
7. Try again

Solution 2: Check Your IP
--------------------------
Run in CMD:
> ipconfig

Find your IPv4 address.
If it's NOT 192.168.1.11:
- Update env.txt with correct IP
- Update batch files with correct IP
- Restart servers

Solution 3: Use Localhost
--------------------------
Instead of IP, use:
http://localhost:3000

This ALWAYS works (no flags needed!)
