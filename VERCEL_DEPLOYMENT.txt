üîå STEP 2: Deploy Socket Server
========================================

Socket server CANNOT run on Vercel (needs persistent connections).
Deploy to Railway or Render (FREE):

OPTION A: Railway (Recommended)
--------------------------------
1. Go to: https://railway.app/
2. Sign up with GitHub
3. Click "New Project" ‚Üí "Deploy from GitHub repo"
4. Select your repo
5. Railway will auto-detect Node.js
6. Add Start Command: node socket-server.js
7. Add Environment Variables:
   - Copy all from env.txt
8. Deploy!
9. Copy the Railway URL (e.g., https://your-app.railway.app)
10. Update in Vercel: NEXT_PUBLIC_SOCKET_URL="https://your-app.railway.app"

OPTION B: Render
----------------
1. Go to: https://render.com/
2. Sign up with GitHub
3. New ‚Üí Web Service
4. Connect your repo
5. Settings:
   - Build Command: npm install
   - Start Command: node socket-server.js
   - Environment: Node
6. Add Environment Variables (from env.txt)
7. Deploy!
8. Copy Render URL
9. Update in Vercel

OPTION C: Heroku (Paid)
------------------------
Similar to Railway/Render but requires credit card.
